README FIRST:

This project is made to run in turboc environment, It may not run on any other environment.

To run this project copy this whole directory to the bin sub-folder of your turboc directory.

Now open the turboc -> Open the Project tab then select this folder and inside that open "PRJ2.PRJ" in that ->

Then the run the Project

This is a nominal project in which uses user defined header files and database via file handling.

If you find any issues or errors or if you think there is a room for improvement then you are welcome to give that feedback.
